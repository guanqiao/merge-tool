# Screenshots

## Main Window

![Main Window](screenshots/main_window.png)

The main window shows:
- Left panel: Directory tree for original files
- Right panel: Side-by-side diff view
- Menu bar with File, Edit, and Help menus
- Toolbar with common actions
- Status bar at the bottom

## File Comparison

![File Comparison](screenshots/file_comparison.png)

Features:
- Side-by-side view with synchronized scrolling
- Color-coded differences:
  - Green: Insertions
  - Red: Deletions
  - Yellow: Modifications
- Line numbers for easy reference
- Current line highlighting

## Directory Comparison

![Directory Comparison](screenshots/directory_comparison.png)

Directory tree features:
- Color-coded panels (blue for left, green for right)
- Double-click files to compare
- Visual indicators for:
  - Modified files
  - Files only in one directory
  - Identical files

## Merge Operations

![Merge Operations](screenshots/merge_operations.png)

Merge controls:
- Copy individual changes between panes
- Navigate through differences
- Undo/redo support
- Save merged result with backup
